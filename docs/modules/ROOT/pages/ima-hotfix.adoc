== Change .env file

Change `CONTAINER_CONFIGS_STREAM` to the {skale-node} hotfix release

[source, shell, subs="attributes"]
----
...
CONTAINER_CONFIGS_STREAM={CONTAINER_CONFIGS_STREAM}
...
----

== Update nodes

[source, shell]
----
skale node update .env
----

More information about the hotfix can be found https://skale.network/blog/audits-for-the-win-the-value-of-consistent-audits/[here]
