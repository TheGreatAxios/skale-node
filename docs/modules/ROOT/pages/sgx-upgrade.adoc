== SGX upgrade procedure

=== Switch to run_sgx directory
**Terminal Command:**

```shell
cd run_sgx
```

=== Stop sgx server
**Terminal Command:**

```shell
docker-compose stop
```

=== Update sgxwallet container version

In `docker-compose.yml` change `image` field to `{sgxwallet-container}`

=== Pull new image
**Terminal Command:**

```shell
docker-compose pull
```

=== Recreate sgxwallet container
**Terminal Command:**

```shell
docker-compose up -d
```
